# security.md

## Password Policy
Passwords must be at least 12 characters and rotated every 90 days. Reusing any of the last five passwords is not permitted.

## MFA
Multi-factor authentication is mandatory for all company systems. Hardware security keys are issued to employees handling production access.

## Data Classification
Company data is classified as Public, Internal, or Restricted. Restricted data, such as customer PII, may only be accessed on a need-to-know basis.

## Incident Reporting
Suspected security incidents must be reported to the security team within one hour of discovery via the incident hotline.
